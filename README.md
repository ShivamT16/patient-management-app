# Patient Management System
The Patient Management Application is designed to streamline the management of patient information within a healthcare facility. It offers tools for healthcare administrators and staff to efficiently manage patient records, view ward-specific data, and access hospital-wide statistics.

#### Live Link - https://patient-management-app.netlify.app/
#### Backend Code - https://github.com/ShivamT16/assignment-21

## Features-

### Hospital View
<li> Users can view complete hospital statistics including total number of specialty wards, patients count , and current occupancy.

### Patient View
<li> Users can access and view the complete medical records of all patients, along with their relevant details.
<li> The administration can Create, Update, or Delete any record.

### Ward View
<li> Users can access and view the complete list of all speciality wards, along with their information.
<li> The administration can Create, Update, or Delete any record.
